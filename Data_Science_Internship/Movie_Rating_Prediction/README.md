# Movie Rating Prediction

This project predicts movie ratings using machine learning techniques as part of the CodSoft Data Science Internship.

## Dataset
The dataset contains information about movies such as genre, director, lead actor, and rating.

## Model Used
- Linear Regression

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

## Result
The model predicts movie ratings with an average error of around 1 rating point.
