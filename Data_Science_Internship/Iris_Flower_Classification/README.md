Iris Flower Classification

This project classifies Iris flowers into different species using machine learning techniques as part of the CodSoft Data Science Internship.

Dataset

The dataset contains flower measurements such as sepal length, sepal width, petal length, and petal width.

Model Used

Logistic Regression

Tools & Technologies

Python
Pandas
Scikit-learn

Evaluation Metrics

Accuracy
Precision
Recall
F1-score

Result

The model successfully classified Iris flowers with 100% accuracy on the test dataset.
